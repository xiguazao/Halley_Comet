执行力培养：
  1. 在项目进行中产生意见或代码上等分歧，要首先服从团队安排
  2. 要按时保质完成个人负责的功能模块的开发，有问题及时沟通
  3. 一切开发要先有简单开发文档或伪代码，凡事都先问自己几个为什么

代码风格：
  1. '='/'+='/'==' 等运算（比较）符号两边都要有空格
  2. ':' 前面要有空格
  3. ',' 后面都要有空格
  例如：
a = 0

for i in range(5) :

    a += 1
    
    i += 10
    
    print a, i
